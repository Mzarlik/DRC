# Reglas Personalizadas del Proyecto DRC

- **Histórico de Planes de Implementación:** Cada vez que se genere y apruebe un `implementation_plan.md` (o cuando se finalice una fase de implementación importante), asegúrate de copiar una versión de ese plan dentro del directorio `docs/planes_implementacion/` con un nombre descriptivo (ej. `fase_7_reportes.md`). Esto es para mantener el contexto histórico de los planteamientos arquitectónicos.
